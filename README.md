# Line - Real-time Sketching Application

Line is a real-time sketching application developed using Next.js, React, Redux, Tailwind CSS, and Socket.IO. It provides users with a platform to create and share sketches collaboratively in real-time.

## Features
- **Real-time Collaboration**: Multiple users can sketch simultaneously and see each other's changes in real-time.
- **Responsive Design**: The application is designed to work seamlessly across various devices and screen sizes.
- **User Authentication**: Secure user authentication ensures that only authorized users can access the sketching features.
- **Persistent Sketches**: Sketches are saved automatically, allowing users to revisit and edit their creations later.
- **Customizable Brush Settings**: Users can customize brush sizes, colors, and other settings to suit their preferences.
- **Export and Share**: Users can export their sketches in various formats and share them with others.
